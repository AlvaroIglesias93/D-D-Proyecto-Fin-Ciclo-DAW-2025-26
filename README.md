# D-D_Proyecto_Fin_Ciclo

El presente Proyecto tiene como objetivo el desarrollo de una plataforma web dedicada a la
consulta y gestión de información para jugadores de Dungeons & Dragons 2024 (D&D 2024), con
una ambientación basada en el universo de The Elder Scrolls.

Importar el .sql de la raíz del proyecto para el correcto funcionamiento del mismo.
